<Dynamic Programming with Three Poitners> Ugly Number

2021-01-23

Link:https://leetcode.com/problems/ugly-number-ii/

Brief Description

Write a program to find the n-th ugly number, the positive numbers whose prime factors only include 2, 3, 5.

Algorithm Analysis

Use an array nums[] and three pointers int i2, i3, i5. Each time compare nums[i2] * 2, nums[i3] * 3 and nums[i5] * 5, add the smallest
one into the array and add the corresponding pointer by 1.
