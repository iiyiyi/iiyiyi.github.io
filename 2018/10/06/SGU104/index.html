<DP>SGU104 Little Shop of Flowers

2018-10-06 • Tag: DP

Link:https://codeforces.com/problemsets/acmsguru/problem/99999/104

Brief Description

Input the number of the bunches of flowers $F$, the number of vases $V$, the aesthetic value $A_{i,j}$ obtained by putting the flower bunch $i$ into the vase $j$. The vase(flower) $i$ must be to the left of the vase(flower) $j$ whenever $i<j$. Output the the maximum sum of aesthetic value and the arrangement as a list of $F$ numbers, so that the $k_{th}$ number on this line identifies the vase in which the bunch $k$ is put.

Algorithm Analysis

Obviously, the problem examines the basic application of dynamic programming.
