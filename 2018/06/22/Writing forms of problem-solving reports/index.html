Title

Example:[UOJ#1] A+B problem

There is a fixed format of the title, which is composed of the source of the problem(such as a contest or an online jude etc), the problem number, and the title of the problem.

Link

Example:http://uoj.ac/problem/1

I will also provide the links to the problems so that audience can read the complete description of the problem and sumbit their codes if the problem comes from an OJ.

Brief Description

Example: Input two integers a and b$ x \leq y $. Output their sum.

Since it’s rather time-consuming to read the lengthy description of the original problem, I will use simple technical terms to interpret it. This part will include the necessary information of the problem, the input (including the date ranges), and the output.

Algorithm analyse

This is the most important part of my problem-solving report. I will describe the main idea of solving process and the algorithms and databases used during coding.
