<Hungarian Method>Luogu 1963 Sequence Transformation

2020-12-30 • Category: Graph - Bipartite Matching • Tag: Hungarian Method

Link:https://www.luogu.com.cn/problem/P1963

Brief Description

There is a sequence $0, 1, 2,\ldots, n-1$. The question asks us to find a permutation of this sequence such that the the inter-ring minimum euclidean distance
between the orignal position and current position of $i$ is a given $d(i)$. If there are several such permutations, find the one with the minimum
lexicographical order.

Algorithm Analysis

If we do not consider the lexicographical order, this is a basic bipartite matching problem and we can use Hungarian Method to solve it.
Now think about the principle of the Hungarian Method. Obviously the latter nodes have the priority to choose the matching they want. For the same node, the edge we search first is more likely to be in the matching. We can
modify the algorithm based on these two ideas: when constructing the bipartite matching, put the smaller position at first; then find the augment paths from $n-1$ back to $0$.
