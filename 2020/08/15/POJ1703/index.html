<Union-find Set> POJ1703 Find them, Catch them

2020-08-15 • Category: LeetCode

Link: http://poj.org/problem?id=1703

Brief Description

There are two gangs in the city. Either state that two criminals belong to different gangs or query whether they are in the same gang.

Algorithm Analysis

The problem examines the basic application of union-find set. There is an acient proverb that ‘The enemy of my enemy is my friend and the friend of my enemy is my enemy’. We can manully constuct $numCriminals$ criminals and mark them as $numCriminals…2numCriminals-1$, where $x+numCriminals$ represents the enemy of $x$. Therefore, for two criminals $i$ and $j$,
(1) If $i$ and $j$ belong to different gangs, union $(i,j+numCriminals)$ and $(j,i+numCriminals)$.
(2) If $i$ and $j$ belong to the same union-find set, $i$ and $j$ belong to the same gangs. If $i$ and $j+numCriminals$ belong to the same union-find set, $i$ and $j$ belong to differnt gangs. Otherwise, we cannot decide whether they belong to the same set.
