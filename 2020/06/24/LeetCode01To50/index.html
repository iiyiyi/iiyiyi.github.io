Problem 5: Longest Palindromic Substring

https://leetcode.com/problems/longest-palindromic-substring/
Related Topics: Dynamic Programming
Solution: $palindrome[i][j]$ denotes whether the substring from position $i$ to position $j$ is palindromic. Obviously, $palindrome[i][i] = true$ and $palindrome[i][i+1] = true$ if $s[i] = s[i+1]$. We can easily reach that for $(j-i+1)\ge 3$, $palindrome[i][j] = true$ if and only if $palindrome[i+1][j-1] = true$ and $s[i] = s[j]$.
