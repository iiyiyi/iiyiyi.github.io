<Stack> POJ1363 Rails

2020-06-25 • Category: Data Structure • Tag: stack

Link: http://poj.org/problem?id=1363

Algorithm Analysis

The problem examines the basic application of stack. At first, the pointer $j$ points to the first element of an array that represents a reorganization of the train. For every new-coming coach $i$, push it to the stack. Then pop all elements in the stack which matches what $j$ is pointing to. If the stack is empty and the $j$ has iterated over the array, return True. Otherwise, return False.
For each query, the time complexity if $O(n)$, where $n$ is the number of coaches.
